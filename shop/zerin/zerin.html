Architecture

How it works

Everything runs in your browser. Data is stored in localStorage — no requests ever leave your device. The service worker caches all assets on first load, giving you full offline functionality.

Add it to your iOS or Android home screen and it behaves exactly like a native app: full-screen, no browser chrome, proper safe-area inset handling.

Export your data at any time as a standard CSV. Import from any bank export. You own everything completely.

Compatibility

Runs everywhere.

Built as a Progressive Web App. Works in any modern browser with no installation required.

On iOS, use Safari → Share → Add to Home Screen for the full native experience with offline support and no browser chrome.

On Android, Chrome will prompt you to install automatically. The service worker caches all assets so it loads instantly — even with no connection.
